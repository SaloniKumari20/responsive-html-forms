# HTML Forms Project

This project contains multiple beginner-friendly HTML form designs created using pure HTML.

## 📂 Included Forms

### 1️⃣ Event Registration Form
- Name
- Email
- Phone Number
- Occupation
- Topics of Interest
- Comments

### 2️⃣ Job Application Form
- Name
- Email
- Phone Number
- Resume Upload
- Experience
- Skills
- Highest Education

### 3️⃣ Complex Multi-Section Form
Includes multiple sections for complete user information:

#### Personal Information
- Name
- Email ID
- Phone Number
- Date of Birth
- Gender
- Country

#### Address Details
- Street
- City
- State
- Zip Code

#### Account Details
- Username
- Password
- Confirm Password

#### Preferences
- Hobbies / Interests
- Language Selection
- Notifications Option

#### Additional Information
- Comments / Message
- Submit Button

## 🛠️ Technologies Used

- HTML5

## 🎯 Purpose of Project

This project was created for practicing HTML forms, input types, fieldsets, labels, checkboxes, radio buttons, dropdowns, password fields, file upload, and multi-section form structure.
